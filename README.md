# prod
Administração da Produção - UFMG

## Como usar

1. Clone o repositório: `git clone https://github.com/Lealwbs/prod.git`
2. Navegue até o diretório do projeto: `cd prod`
3. Instale as dependências: `make setup`
4. Execute o programa: `make run`
5. Para rodar os testes: `make test`

## Comandos

- make setup: Instala as dependências necessárias.
- make run: Executa o programa principal.
- make test: Executa os testes.
- make clean: Limpa os arquivos gerados.
- make help: Exibe todos os comandos disponíveis.
